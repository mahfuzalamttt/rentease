# RentEase

A modern furniture and appliance rental platform built with Next.js, TypeScript, and Tailwind CSS.

## Overview

RentEase is a frontend-based rental platform that allows users to browse furniture and appliances, explore product details, save products to a wishlist, add items to a cart, and complete a rental checkout flow.

The project focuses on delivering a clean, responsive, and user-friendly rental experience for students, professionals, and families.

## Features

* Responsive Home Page
* Product Listing Page
* Product Search Functionality
* Category Filtering
* Product Detail Page
* Wishlist Management
* Add to Cart Functionality
* Cart Management
* Checkout Page
* Rental Confirmation Page
* Login & Registration UI
* About Page
* Contact Page

* Responsive Navigation Bar
* Modern Footer Section
* Centralized Product Data Management

## Installation

### Clone the Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open in Browser

```txt
http://localhost:3000
```
